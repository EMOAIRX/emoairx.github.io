з”ҹжҲҗеҮҪж•°е®ҡд№ү

$G(x) = g_0+g_1x+g_2x^2+вҖҰвҖҰ = \sum_{n>=0} g_nx^n$пјҢжҲ‘д»¬з§°$G$жҲ–иҖ…$G(z)$жҳҜж•°еҲ—$<g_0,g_1,вҖҰ>$зҡ„з”ҹжҲҗеҮҪж•°гҖӮ

еёёз”Ёз”ҹжҲҗеҮҪж•°

$$
\sum_{n>=0}[n=0]z^n = 1\
\sum_{n>=0}z^n = \frac{1}{1-z}\\sum_{n>=0}(-1)^nz^n = \frac{1}{1+z}\\sum_{n>=0}\binom{m}{i}z^n = (1+z)^m\\sum_{n>=0}\binom{n+m-1}{n}z^n = \frac{1}{(1-z)^m}\\sum_{n>=1}\frac{1}{n}z^n = \ln\frac{1}{1-z}\\sum_{n>=1}\frac{(-1)^{n+1}}{n}z^n = \ln(1+z)\\sum_{n>=0}\frac{1}{n!}z^n = e^x
$$

еҶҷеҲ°иҝҷйҮҢеҸ‘зҺ°иҮӘе·ұдёҚдјҡ$\ln$иҝҷзҺ©ж„Ҹе„ҝ

$exp$жҳҜжҖҺд№Ҳе®ҡд№үжқҘзқҖзҡ„?$exp(A(x)) = \sum_{i>=0}\frac{A^i(x)}{i!}$

$ln$еҸӘзҹҘйҒ“дёҖдёӘз»“и®әпјҡ$ln(x) = \frac{1}{x}$пјҢжүҖд»Ҙ$ln(f(x)) = \int\frac{fвҖҷ(x)}{f(x)}$пјҹеӨ§жҰӮе°ұжҳҜеҜ№зҡ„еҗ§

жҠҠ$f(x) = e^{g(x)}$д»ЈиҝӣеҺ»пјҢеҸ‘зҺ°$ln(e^{g(x)}) = \int\frac{e^g(x)*gвҖҷ(x)}{e^{g(x)}} = g(x)$

зңӢиө·жқҘйқһеёёзңҹгҖӮе—ҜгҖӮ

жңүдёҖдёӘе°ұжҳҜжұӮеүҚзјҖе’Ңе°ұжҳҜеҚ·з§ҜдёҠ$<1 ,1,1,1,вҖҰ> = \frac{1}{1-z}$

з®ҖеҚ•з”ҹжҲҗеҮҪж•°жұӮжі•

дёҫдёҖдёӘдҫӢеӯҗеҗ§пјҢжҜ”еҰӮиҜҙж–җжіўйӮЈеҲҮж•°еҲ—зҡ„з”ҹжҲҗеҮҪж•°пјҢеӣ дёәжңү$f(x)=f(x-1)+f(x-2)$жңү$F = xF + x^2F + 1$пјҢжүҖд»ҘжҳҜ$F = \frac{1}{1-x-x^2}$

и§ЈйҖ’еҪ’ејҸ

еӨ§жҰӮе°ұжҳҜзҹҘйҒ“з”ҹжҲҗеҮҪж•°жұӮйҖҡйЎ№е…¬ејҸпјҢзңӢиө·жқҘеңЁеҒҡж•°еҲ—йўҳзҡ„ж—¶еҖҷеҸҜиғҪдјҡжңүзӮ№з”Ёпјҹ

дёҖзұ»зү№ж®Ҡзҡ„з”ҹжҲҗеҮҪж•°

еӨ§жҰӮжҳҜеҲҶжҜҚеҸҜд»ҘиҝӣиЎҢйғЁеҲҶеҲҶејҸеҲҶи§Јзҡ„

дёҖдёӘйңҖиҰҒиҜҒжҳҺзҡ„дёңиҘҝпјҡ

$R(z) = \frac{P(z)}{Q(z)}$пјҢе…¶дёӯ$P$,$Q$жҳҜеӨҡйЎ№ејҸпјҢз„¶еҗҺиҰҒжұӮRзҡ„з¬¬nйЎ№зҡ„зі»ж•°гҖӮ

ж №жҚ®еҲҡжүҚзҡ„дёңиҘҝпјҢжҲ‘д»¬зҹҘйҒ“
$$
\frac{1}{(1-px)^{m+1}} = \sum_{n>=0}\binom{m+n}{m}p^nz^n
$$
иҝҷдёңиҘҝеҸҜд»Ҙеҝ«йҖҹз®—е®ғзҡ„дёҖдёӘдҪҚзҪ®дёҠзҡ„еҖјгҖӮ

з„¶еҗҺеҰӮжһңжҳҜиӢҘе№ІдёӘдёҠйқўзҡ„дёңиҘҝзҡ„е’Ң
$$
\frac{a_1}{(1-p_1x)^{m_1}} + \frac{a_2}{(1-p_2x)^{m_2}}+вҖҰ
$$
д№ҹжҳҜеҫҲеҘҪжұӮзҡ„пјҢеҸӘйңҖиҰҒжҠҠжҜҸдёҖйЎ№зҡ„зі»ж•°еҠ иө·жқҘе°ұиЎҢдәҶгҖӮ

з„¶еҗҺеҸҜд»ҘиҜҒжҳҺеҜ№дәҺ$R(0) \not= \infty$зҡ„жңүзҗҶеҮҪж•°$R(x)$йғҪеҸҜд»ҘиЎЁзӨәжҲҗ$R(x) = S(x) + T(x)$пјҢе…¶дёӯSжҳҜдёҠйқўйӮЈдёӘеҪўејҸпјҢTжҳҜдёҖдёӘеӨҡйЎ№ејҸгҖӮгҖҗдёҚзҹҘйҒ“дёәд»Җд№ҲпјҢдёҚдјҡиҜҒжҳҺгҖ‘

жүҖд»Ҙдё»иҰҒжҳҜиҰҒйғЁеҲҶеҲҶејҸеҲҶи§ЈгҖӮ

е…ҲиҖғиҷ‘еҰӮдҪ•жҠҠдёӢйқўзҡ„йӮЈдәӣ$p_i$жұӮеҮәжқҘгҖӮ

еҒҮеҰӮ$Q(x) = q_0+q_1x+вҖҰ+q_mx^m | q_0\not=0,q_m\not=0$

жҲ‘д»¬д»Ө$Q^R(x) = q_0x^m+вҖҰ+q_m$

жңү$Q^R(x) = q0(x-\rho_0)(x-\rho_1)вҖҰ(x-\rho_m)$

жңү$Q(x) = q0(1-\rho_0x)(1-\rho_1x)вҖҰ(1-\rho_mx)$

$$

$$
