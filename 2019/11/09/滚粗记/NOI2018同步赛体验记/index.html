Day2 $LOJ$上的得分是85+30+25。

加上50分笔试分。差银牌线7分。

「NOI2018」归程

25分做法：求出最短路后直接可持久化并查集维护。

100分做法：初始化离散化数组。

「NOI2018」你的名字

68分做法

​ 建出广义后缀自动机。

​ 求出最后出现位置，暴力跳后缀链接。

​ 时间复杂度$O(n \sqrt n)$

​ 常数很小。

56分做法：

​ 把前面直接维护的东西改成线段树合并爆right集合，并判断。（搏一搏单车变空气）

​ 时间复杂度$O(n \sqrt n log (n))$

​ 常数很小。但还是要5~6s。

加一句如果已经找到直接break的优化……
